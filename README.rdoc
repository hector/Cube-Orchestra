= Cube Orchestra

This is the final project for the Advanced Interface Design subject
for the masters {SMC}[http://www.upf.edu/smc/]
and {CSIM}[http://csim.upf.edu/] from Pompeu Fabra University.

It is a continuation of the work done at the previous project {rtifinal}[https://github.com/hector/rtifinal].
It will be presented on the 9th June 2012 at the {8a Mostra Sonora i Visual}[http://www.conventagusti.com/?p=797]
del Convent de Sant Agustí, Barcelona.
